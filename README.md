# Fullstack Emotion Detector

🚀 Full stack web app that detects emotions from user-input text using Hugging Face API.

## 📦 Tech Stack
- Backend: Spring Boot
- Frontend: Angular
- AI: Hugging Face Transformers API
- Cloud: AWS (ECS + S3)
- Dockerized: Yes
